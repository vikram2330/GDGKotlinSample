# GDG DevFest Hyderabad 2019 - Kotlin Android Sample

Sample project to showcase the usage of basic kotlin features in Android.

## Some kotlin features demonstrated
- Data classes
- Coroutines
- Extension Functions
- Sealed Classes

## Installation

```bash
./gradlew clean assembleDebug
```
